# Xingshu Sidebar Image2 Icon Kit v1

侧栏专用 Image2 图标套件，针对 `24-30px` 导航尺寸做了更粗线条和更少细节的处理。

- 生成方式：Image2 生成 3x3 图标表，使用本地 chroma-key 去底色后切分。
- 使用背景：透明 PNG。
- 使用范围：侧栏导航、新建对话按钮、更多分组、数据资产子菜单。
- 视觉约束：蓝色线性、圆角端点、少量青色节点、无星轨背景、无 3D、无 emoji、无彩色渐变底板。
