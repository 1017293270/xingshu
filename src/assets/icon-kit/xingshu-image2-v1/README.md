# Xingshu Image2 Icon Kit v1

Generated with the built-in image2 workflow as a first-pass unified product icon kit.

## Style

- Blue and cyan enterprise SaaS line icons.
- Transparent PNG outputs.
- No text baked into the icons.
- No card backgrounds in the icon files.
- No forced orbit ring around every icon.

## Files

- `xingshu-icon-kit-v1-sheet-source.png`: original chroma-key sheet.
- `xingshu-icon-kit-v1-sheet-transparent.png`: transparent sheet after local background removal.
- `icon-*.png`: 256x256 transparent icon assets.
- `manifest.json`: icon ids, Chinese labels, and filenames.
